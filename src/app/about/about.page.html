Détails et caractéristiques Détails du projet Description du Projet Ce projet est une application CRUD de gestion des Pokémon, développée avec Ionic et Angular. Il permet de lister, créer, modifier et supprimer des Pokémon tout en offrant une interface utilisateur intuitive et responsive. Fonctionnalités principales
  • Affichage de la liste des Pokémon avec pagination.
  • Ajout de nouveaux Pokémon avec des caractéristiques personnalisées.
  • Modification des informations des Pokémon existants.
  • Suppression d'un Pokémon de la liste.
  • Recherche et filtrage par nom ou type.
 Technologies utilisées
  • Ionic Framework : pour créer une interface mobile hybride.
  • Angular : pour la gestion des composants et des données.
  • TypeScript : pour un code robuste et typé.
  • API REST : pour la gestion des données des Pokémon.
 Étapes de développement
  1. Initialisation du projet avec Ionic.
  2. Création des composants de l'application (liste, détail, formulaire).
  3. Implémentation des fonctionnalités CRUD.
  4. Intégration avec une API REST pour la gestion des données.
  5. Tests et optimisation de l'interface utilisateur.
